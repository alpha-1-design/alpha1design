# Changelog

All notable changes to Alpha-1 Design will be documented in this file.

## [1.0.0] - 2024-04-16

### Added
- **Color Palette Generator**
  - 6 harmony modes (Analogous, Complementary, Triadic, Split Comp, Tetradic, Grayscale)
  - Gradient mode with start/end colors
  - WCAG contrast checker (AA/AAA ratings)
  - Extract colors from image
  - Share via URL
  - Export to CSS, SCSS, JSON, Tailwind
  - Save/load palettes
  - Lock individual colors

- **Image Tools**
  - Image compressor with quality slider
  - Resize presets (1920x1080, 1280x720, 800x600, etc.)
  - Before/After comparison slider
  - Image metadata display
  - Collage/Moodboard creator

- **AI Writer**
  - Groq API (free, no key required)
  - Streaming responses
  - Prompt templates
  - User-configurable API keys (Claude, Gemini, Mistral)

- **UX Features**
  - Glassmorphism effects
  - Dark/Midnight/Light themes + System auto-detect
  - Command Palette (⌘K)
  - Toast notifications
  - Offline indicator
  - Loading skeletons
  - Keyboard shortcuts page
  - PWA support

- **Meta**
  - SEO meta tags
  - README with full feature list
  - Code of Conduct
  - Contributing guide
